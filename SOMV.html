Configuracion de un sistema Operativo

La configuración es un conjunto de datos que determina el valor de algunas variables de un programa o de un sistema operativo.
De forma general, la instalación se compone de los siguientes pasos:

  • Configuración del dispositivo de arranque en la BIOS
  • Creación de las particiones en el disco duro
  • Creación del sistema de ficheros
  • Selección de paquetes 
  • Configuración 
  • Instalación y configuración:

En el caso concreto de a práctica, se va a instalar un sistema operativo Microsoft Windows. El instalador se subdivide en dos partes fundamentales:

  • Instalación modo texto: Se encarga de detectar y cargar los controladores necesarios. Permite crear las particiones y sistema de ficheros y finaliza realizando la copia de los ficheros imprescindibles.
  • Instalación gráfica: encargada de realizar la copia de los ficheros del sistema y aplicaciones del sistema operativo y permite configurar las opciones de internacionalización y de red.

Es importante que realices ajustes el sistema operativo para hacerlo más seguro como:

  • Deshabilita las carpetas compartidas si no las utilizas. Esto evita la propagación de programas maliciosos que las aprovechen para infectar el ordenador.
  • Utiliza contraseñas seguras y fáciles de recordar tanto en aplicaciones como a nivel de acceso al ordenador para evitar que puedan acceder personas no deseadas.
  • Crea perfiles de usuario con privilegios restringidos, de manera que se limiten las acciones de algunos usuarios que puedan provocar un aumento de posibilidades de infección.
  • Deshabilita la ejecución automática de dispositivos de almacenamiento extraíbles (como USB), ya que pueden contener aplicaciones maliciosas que se ejecuten en segundo plano, invisibles al usuario.
  • Ten en cuenta que el soporte técnico en versiones antiguas de sistemas operativos y aplicaciones recibe menos atención que en el de las últimas versiones, por lo que por norma general las versiones más antiguas están más expuestas a vulnerabilidades.
  • Normalmente los archivos maliciosos se esconden en el sistema como ficheros ocultos, por lo que muchas veces se encuentran configurando el sistema para que se permitan ver los archivos ocultos.
  • Es posible configurar la visualización de las extensiones de archivos para que puedas identificar las extensiones de archivos que se hayan descargado y evitar ser víctima de técnicas como la doble extensión.


 

PRACTICA: Configuración de un sistema operativo
Para la práctica se realizaron los siguientes pasos:

  • Instalación del Sistema Operativo (vídeo explicativo)
  • Personalizar el explorador de ficheros. Para abrir el explorador de archivos se puede o bien hacer doble click sobre el icono mi PC o bien mediante la combinación de teclas: “tecla de Windows + E”. Una vez abierto, seleccionar la opción de menú: Herramientas → Opciones de carpeta:
  • Desactivar «Ocultar extensiones de archivos conocidos». En la pestaña Ver, dentro de la parte Configuración avanzada.
  • Desactivar «Recordar configuración de vistas de carpetas». En la pestaña Ver, dentro de la parte Configuración avanzada.
  • Instalar Style XP (está en la unidad compartida C:\) y personalizar el escritorio)
  • Probar como configurar la barra de exploración, estilos visuales, el fondo de pantalla, transparencias, etc.
  • Comprobar la correcta configuración de la red
  • Realizar una consulta para resolver la dirección www.fi.up.es. Para ello, abriremos una ventana de símbolo de sistema, bien mediante la opción del menú Inicio, Ejecutar → cmd, o bien en el menú Inicio, Programas → Accesorios → Símbolo de sistema.
  • Realizar un ping a www.fi.up.es para comprobar la conectividad. En la misma ventana de símbolo de sistema, teclear la orden ping www.fi.up.es. De este modo, podremos ver si el servidor web de la facultad está activo o incluso si nuestro ordenador tiene conectividad con el exterior.
  • Navegador
  • Descargar e instalar el navegador Maxthon de http://www.maxthon.com.
  • Descargar la traducción al español de la misma página (sección downloads). 
  • Configurar el navegador. Se aconseja probar las siguientes opciones, utilizando el asistente de configuración (Open Configuration Wizard).
  • Habilitar Súper Arrastrar y Soltar
  • Habilitar Control con ratón
  • Bloqueo completo de elementos emergentes y publicidad de las páginas.
  • Abrir el centro de configuración de Maxthon (Open Maxthon Setup Center). En esta ventana podemos configurar múltiples opciones del navegador.
  • Visitar alguna página con publicidad, por ejemplo www.terra.es o www.elpais.es y veremos como esta se suprime automáticamente.
  • Probar las «acciones de ratón»
  • Añadir un par de fuentes a la sindicación de contenidos disponible en el navegador Maxthon, en la parte izquierda de la pantalla, bajo el nombre Mis suscripciones.
  • Búsquedas web: realizar las búsquedas y comentar las diferencias entre ellas
  • Sintaxis básica: buscar «iglesias en julio» y «“iglesias en julio”»
  • Opciones avanzadas web mediante google.
  • Excluyendo palabras clave: excluir «“julio iglesias”» de cada una de las búsquedas anteriores
  • Dentro de un sitio: Realizar las dos búsquedas sólo para fi.upm.es
  • Páginas que enlazan a una dada: Repetir las búsquedas en las páginas que enlacen a una cualquiera de las obtenidas en las búsquedas anteriores
  • Páginas relacionadas con una: Buscar páginas relacionadas con una de las páginas de cualquier búsqueda anterior.
  • Correo electrónico utilizando Microsoft Outlook (hay que solicitar la información en la sala de operadores con identificación)
  • Configurar una cuenta IMAP
  • Es posible la configuración de una cuenta POP30 (Hay que tener en cuenta que el correo existente se descargará al disco duro local borrándose del servidor)
  • Configurar una cuenta HTML (Necesaria una cuenta de Hotmail)
  • Crear vuestra tarjeta de visita y añadidla como firma a un mensaje (menú Herramientas → Opciones, en la pestaña Formato de correo)
  • Utilidades necesarias
  • Buscar, instalar y probar al menos 7 utilidades (el asterisco indica las que posiblemente sean más útiles)
